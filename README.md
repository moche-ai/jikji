# Jikji (직지)

**Your memory, owned by you, portable across every AI.**

Jikji is a Korean-first personal memory layer for AI agents — a structured, graph-based
memory system that any AI tool can share through MCP.

> Named after *Jikji*, the world's oldest extant book printed with movable metal type
> (Korea, 1377) — the original technology for making memory permanent.

## What it does

- **One memory, every agent** — Claude Code, Cursor, custom agents and workflows share a
  single personal memory through MCP (gateway & framework adapters on the roadmap).
- **Structured recall** — hybrid retrieval (BM25 + dense + graph + reranker) with
  scope-aware disambiguation across multiple projects.
- **Trust by design** — training use is **off by default**, memories are reviewable before
  saving, fully exportable, and deletion cascades.
- **Korean-first** — retrieval quality measured on a Korean long-term memory benchmark;
  English is first-class, CJK is the edge.
